# Todo App

Basic todo app with simple functionality (finish todo, edit todo, delete todo). You can also filter todos by All, Active, and Finished. The todo list can also be reset. Todo tasks are stored using localStorage.

### Built with

- [Vite](https://vitejs.dev/)
- [React.js](https://react.dev/)
- [React Icons](https://react-icons.github.io/react-icons/)
- [Tailwind CSS](https://tailwindcss.com/)
